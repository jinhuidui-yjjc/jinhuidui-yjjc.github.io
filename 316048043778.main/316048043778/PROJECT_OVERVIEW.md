# 《正义链：检察官的24小时》项目概览

## 项目简介
这是一款名为《正义链：检察官的24小时》的互动式教育游戏，玩家将扮演一名见习检察官，在24小时内处理10个不同类型的法律案件，每个选择都会影响最终的"正义值"评分和能力评估。

## 主要功能
- 10个不同类型的检察业务案例（刑事、民事、行政、公益诉讼、未成年人检察）
- 每个案例提供4个选择，选项顺序随机打乱
- 实时计算用户得分（满分100分）
- 基于答题情况动态生成能力评估报告
- 结果分享功能
- 响应式设计，适配手机屏幕

## 技术栈
- React 18+
- TypeScript
- Tailwind CSS
- Framer Motion（动画效果）
- React Router（路由管理）
- Vite（构建工具）

## 文件结构说明

### 主要目录
- `src/` - 源代码目录
  - `components/` - React组件
  - `lib/` - 工具函数和游戏数据
  - `pages/` - 页面组件
  - `contexts/` - React Context
  - `hooks/` - 自定义React Hooks

### 核心文件说明

#### 1. 入口文件
- `index.html` - 项目入口HTML文件
- `src/main.tsx` - React应用入口文件，配置了Router和Toaster

#### 2. 页面组件
- `src/pages/Home.tsx` - 主页面，管理游戏状态和屏幕切换
- `src/components/CoverScreen.tsx` - 游戏封面页
- `src/components/CaseScreen.tsx` - 案件处理页面
- `src/components/ResultScreen.tsx` - 结果展示页面

#### 3. 游戏数据和工具
- `src/lib/gameData.ts` - 包含10个案例数据和相关类型定义
- `src/lib/utils.ts` - 工具函数，如生成评语、计算评级等

## 核心组件功能介绍

### 1. Home.tsx
- 管理游戏的整体状态（当前屏幕、分数、答题记录等）
- 处理游戏流程控制（开始游戏、切换案件、显示结果等）
- 计算能力评估数据

### 2. CaseScreen.tsx
- 显示当前案件内容和选项
- 处理用户选择
- 随机打乱选项顺序
- 显示选择后的提示信息

### 3. ResultScreen.tsx
- 展示最终得分和评级
- 显示各项能力评估图表
- 生成个性化评语
- 提供分享和重新开始功能

## 游戏流程

1. **开始游戏**：用户在封面页点击"开始挑战"按钮
2. **处理案件**：依次处理10个不同类型的案件，每个案件有4个选项
3. **计算得分**：根据用户选择实时计算得分和各项能力值
4. **查看结果**：完成所有案件后，显示最终得分、评级和能力评估
5. **分享/重玩**：用户可以分享结果或重新开始游戏

## 部署指南

1. 安装依赖：`pnpm install`
2. 开发模式运行：`pnpm dev`
3. 构建项目：`pnpm build`
4. 构建后的文件位于`dist/static`目录，可以部署到任何静态网站托管服务

## 公众号集成说明
游戏构建后，可以通过以下方式集成到公众号：
1. 将构建后的文件部署到服务器
2. 在公众号推文中添加"阅读原文"链接或卡片链接指向游戏URL
3. 用户点击链接即可在微信内置浏览器中玩游戏
